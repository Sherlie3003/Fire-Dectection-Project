# 🔥 Real-Time Fire Detection System


## 📌 Project Overview
This project detects fire in real-time using Python and OpenCV.
When fire is detected, the system automatically sends an email alert.

## 🛠 Technologies Used
- Python
- OpenCV
- NumPy
- SMTP (Email Alert)

## 🚀 Features
- Real-time video processing
- Fire detection using image processing
- Automatic email notification system

## 📷 Input
- Webcam / CCTV Camera feed

## 📧 Output
- Email alert when fire is detected
